# DSpace Release 10.0 Testathon Page

[View edit history] (https://github.com/hostle83/dspace-testathon-demo/blob/main/README.md)
This site documents the testing process for the upcoming testathon for DSpace 10.

## Goal

Test core functionality of DSpace and identify bugs before release.

## How to Participate

1. Go to the GitHub Issues list
2. Pick a test
3. Move it to **In Progress**
4. Execute the test
5. Mark **Passed** or **Failed**
6. If failed, create a bug report

## Example Tests

- Submit a new item
- Search for an item
- Edit metadata
- Approve workflow
